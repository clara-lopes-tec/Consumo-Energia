## 🎯 Objetivo
Este sistema foi desenvolvido para a disciplina de *Desenvolvimento de Sistemas* com o intuito simples de cálcular gastos mensais domésticos.

## 🛠️ Tecnologias Utilizadas
* **Linguagem:** Python
* **Versionamento:** Git & GitHub
* **Ambiente:** Visual Studio Code

## 🧮 Lógica e Fórmula
O sistema utiliza a seguinte fórmula pra chegar no resultado:
$$Consumo(kWh) = \frac{Potência(W) \times Horas \times 30 \text{ dias}}{1000}$$
